# WebScrapingVG

<br>

En este proyecto, realizamos un análisis del mercado al que nos dirigimos. Nuestro objetivo es comprender las tendencias de precios de diversos juegos y llevar a cabo un estudio de mercado que nos permita identificar oportunidades y estrategias efectivas.
